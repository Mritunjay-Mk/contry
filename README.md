# Contry

Contry is a modern Expo + TypeScript expense-sharing app for roommates, friends, trips, offices, and groups.

## Features

- Group, member, expense, settlement, history, scanner, reports, and settings screens
- Zustand state management with automatic split and settlement calculations
- SQLite schema and repository layer prepared for local persistence
- Light and dark Material Design 3 theme using React Native Paper
- Lucide icons, rounded cards, chart analytics, export/share service modules
- Future-ready service boundaries for Firebase sync, Google login, voice entry, UPI settlement, and cloud backup

## Run

```bash
npm install
npm run start
```

Use Expo Go or a simulator from the Expo developer tools.
